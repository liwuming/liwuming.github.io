常用方法 getYear, getMonthValue,获取月份，返回值int类型，返回值范围1-12 getDayOfMonth,获去月份中的日期，返回值int类型，1-31 getDayOfWeek,获取星期几，返回值dayOfWeek类型，是一个枚举类型

对于国人来说，更期待getDayOfWeek方法可以返回一个int类型的值，可以getvalue方法

//默认时区的系统时钟的当前时间

System.out.println(now);

var weekday = now.getDayOfWeek().getValue();
System.out.println("weekday:"+weekday);
