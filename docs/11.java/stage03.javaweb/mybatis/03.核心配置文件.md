# 类型别名

![image-20240305071127103](C:/Users/14595/AppData/Roaming/Typora/typora-user-images/image-20240305071127103.png)

```xml
<typeAliases>
  <package name="net.ibiancheng"/>
</typeAliases>
```



添加类型别名,自动包扫描,并且可以忽略大小写





> 注意,标签的前后顺序,这是xml文件本身的限制
