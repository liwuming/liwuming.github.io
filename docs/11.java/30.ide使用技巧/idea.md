
IDEA中快速跳出引号，中括号，大括号的快捷方式


方式一：

①、鼠标在括号内，要跳到括号外面，并负责添加分号。

快捷键：ctrl+shift+enter

这个操作叫做：Complete Current Statement（完成当前语句）

效果：如果当前语句后面没关闭，会自动帮你关闭。（即在当前语句中加入分号）

如果有关闭了（即有分号），会直接跳至下一行。

②、鼠标在括号内，直接跳转至下一行。

快捷键：shift enter

效果：不管后面用没有关闭都直接跳至下一行
方式二：

①、跳出双引号：shift + "

②、跳出单引号：’

③、跳出括号：shift + )

④、跳出中括号：]

以此类推。
