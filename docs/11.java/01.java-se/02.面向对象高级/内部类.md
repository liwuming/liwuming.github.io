ArrayList泛型数组

add，在数组列表的尾部添加新的元素
size，返回数组列表包含元素的个数
trimToSize,将存储块的大小调整为保存当前元素数量所需要的存储空间

> 应该在确认不会再向数据列表中添加任何元素时再调用trimToSize方法
