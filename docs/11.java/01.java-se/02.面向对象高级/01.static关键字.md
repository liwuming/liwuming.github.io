# 如何引用静态成员变量
可通过类名.成员变量的方式来引用静态成员变量，但在同一类中，包括继承中，可省略类名。






被static关键字修饰的变脸




类方法，
类方法不依赖实例状态
类方法仅依赖静态属性

类方法的一个较为常用的场景就是工具类

因为静态方法属于类，而不属于对象的实例，因此不需要实例化就可以使用，且由于不需要实例化，因此可以节省内存开销。
