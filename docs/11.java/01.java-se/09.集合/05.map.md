put,，向集合中添加key-value,如果key已经存在，则会覆盖原有值。
get,通过key获取value，如果key不存在，则返回null，但这通常并不友好，如果key不存在希望为其设定个默认值，则可以使用getOrDefault(key,default)

size,获取集合中元素的个数
containsKey,判断key是否存在，
containsValue,判断value是否存在
remove,根据key移除元素，
clear,清空集合中的所有元素
