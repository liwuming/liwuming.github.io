


## 第一个go程序

go源文件是以.go为后缀的文件,新建一个hello.go文件,写入如下内容:

```go title="hello.go"
package main

import "fmt"

func main() {
	fmt.Println("hello go world")
}
```

执行go build hello.go,会生成一个hello.exe文件,然后执行hello
