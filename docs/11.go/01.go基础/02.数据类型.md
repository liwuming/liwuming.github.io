# 数组与切片

go语言的数组的定长性和值拷贝限制了其使用场景。为了弥补数组的定长性不足，go提供了另一种数据类型slice，译为切片，是一种变长数组，其数据结构中有指向数组的指针，所以是一种引用类型。




数组与切片的区别
1. 数组是定长的，切片是变长
2. 数组是值类型，切片是引用类型



切片的属性
len，元素长度，
cap，容量

`len<=cap`,


```go title="src/slice.go"
type slice struct {
	array unsafe.Pointer
	len   int
	cap   int
}
```
