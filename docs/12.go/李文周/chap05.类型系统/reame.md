### 为类型添加方法

类型别名
为什么需要类型别名,

go语言虽然是面向对象语言,但是并不是像传统的面向对象语言那样有类的概念,


可以给类型添加方法,为了避免污染全局的类型,因此可以给


在go语言中,可以给任意类型添加相应的方法.

```go
// 定义类型别名
type Integer int

func (a Integer) less(b Integer) bool {
	return a < b
}

func main() {
	//在使用类型别名时,不能使用类型推断,需要显式的指定变量的类型
	var num1 Integer = 10
	num2 := 10

	fmt.Printf("%T,%T", num1, num2)
	fmt.Println(num1.less(20))
	//fmt.Println(num2.less(20))
}
```

在本例中声明了一个int类型的别名Integer,既然是别名,所以Integer类型和int类型没有本质的不同,只是为了给int类型增加less方法,而不污染其他int类型而已.

注意事项:
在使用类型别名自定义变量时,必须显式的指定变量类型,不可以使用类型推断,

类型推断仅用于局部变量,且基本类型


扩展

go语言中的面向对象最为直观,也无需支付额外的成本,如果要求对象必须以指针传递,又是会是额外成本,

> 只有在需要修改对象时,才必须用指针,这不是go语言的约束,而是一种自然的约束.


关于指针的两个操作

获取指针的值,*,

究其原因,go语言和c语言一样,类型都是基于值进行传递的,要想修改变量的值,只能传递指针.







### 值语义和引用语义

值语义和引用语义的区别在于赋值,比如下面例子:

```go
b=a
b.modify()
```

如果b的修改不会影响a的值,那么此类型属于值类型,如果会影响a的值,那么此类型属于引用类型.

# 切片与数组的区别


数组与切片之间的关系


数组,
具有确定数量的若干个相同数据类型的元素的序列.

```go
func main() {
	//使用数组字面量来初始化一个数组
	//nums1 := [3]int{1, 2, 3}
	nums1 := [...]int{1, 2, 3}

	for i, num := range nums1 {
		fmt.Println(i, num)
	}
}
```

在数组字面量中,如果省略号出现在数组长度的位置,那么数组的长度由初始化数组的元素个数决定,以上代码可以简化为.
```go
nums1 := [3]int{1, 2, 3}
```

```go
nums1 := [...]int{1, 2, 3}
```


切片看上去像是没有长度的数组类型.

数组和切片是紧密关联的,slice是一种轻量级的数据结构,可以用来访问数组的部分或者全部元素.

slice有三个属性,指针,长度和容量
