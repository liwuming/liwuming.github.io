常量计数器iota

iota在const关键字出现时被重置为0，
const关键字每出现一次计数累加一次，
如果两个const的赋值语句的表达式是一样的，那么可以省略后一个赋值表达式，

```go
func main()  {
	const (
		a1=iota
		b1=iota
		c1=iota
	);

	const (
		a2=1<<iota
		b2=1<<iota
		c2=1<<iota
	);

	/*
	左移运算，
	1<<iota,1<<0,1*2^0=1
	1<<iota,1<<1,1*2^1=2
	1<<iota,1<<2,1*2^2=4
	
	*/

	//a=0,b=1,c=2
	fmt.Println("a=%v,b=%v,c=%v",a1,b1,c1);
	fmt.Println("a=%v,b=%v,c=%v",a2,b2,c2);
}
```
