## 参数列表

在 go 语言中，函数参数是不支持默认值的
如果多个参数的类型一样，可以进行简化

1. go语言中的函数参数不支持默认值
2. 所有参数都是按照值进行传递的
3. 变长参数


变长参数

所谓变长函数，是指函数可以接受的参数列表长度不固定。如最常用的fmt.Println,

```go
func main() {
	fmt.Println(sum(1, 2))
	fmt.Println(sum(3, 4, 5))
}

func sum(nums ...int) int {
	total := 0
	for i, num := range nums {
		fmt.Println(i, num)
		total += num
	}
	return total
}
```
